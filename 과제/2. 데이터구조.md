3.  다음은 여러 사람의 혈액형(A, B, AB, O)에 대한 데이터이다. 반복문을 사용하여
   key는 혈액형의 종류, value는 인원 수인 딕셔너리를 만들고 출력하시오.

   ```python
   blood_types = ['A', 'B', 'A', 'O', 'AB', 'AB', 'O', 'A', 'B', 'O', 'B', 'AB']
   a = 0
   b = 0
   ab = 0
   o = 0
   for bl in blood_types:
       if bl == 'A':
           a += 1
       elif bl == 'B':
           b += 1
       elif bl == 'AB':
           ab += 1
       elif bl == 'O':
           o += 1
   print({'A':a,'B':b,'AB':ab,'O':o}) 
   ```

   ```python
   blood_types = ['A', 'B', 'A', 'O', 'AB', 'AB', 'O', 'A', 'B', 'O', 'B', 'AB']
   
   blood_dict = {}
   
   for bl in blood_types:
       if bl not in blood_dict.keys():
           blood_dict[bl] = 1
       else:
           blood_dict[bl] += 1
   print(blood_dict)
   ```
